# CreateAI

**Grammarly のように複数のアプリを跨いでユーザに付いてくる AI エージェント**を作るプロジェクトです。

既存のローカル LLM（[Ollama](https://ollama.com/)）を「頭脳」として使い、
**思考 → 行動 → 観察（ReAct ループ）**でツールを反復的に使わせることで、
ユーザの自然言語による指示をアプリ境界を越えた一連の操作へと変換します。

## 代表的なユースケース

> 「このファイルを今進めてるプロジェクトにダウンロードして」

→ web からファイルを取得し、**推論で「今のプロジェクト」の尤もらしいディレクトリに配置**します。
